# User Management System - CRUD Application using Node

## Project structure
      App     -
              -   Assets
                        -   Css
                        -   Js
                        -   img
              -   Server
                        -   Model
                        -   Controller
                        -   Database
                        -   Routes
                        -   services
              -   Views
              -   server.js
              
## Usage
      npm install
      
 Create config.env file and create PORT and MONGO_URI Variable and specify Value. 
 To run this project:

      npm start
